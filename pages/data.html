市价单

市价单指在定单接收时以市场上最好的价格执行的定单。

止盈单

止盈单仅当市场价格达到了指定价位或比之更高的时候被执行。止盈卖单是当卖价达到所要求的价位时被执行。止盈买单是当买价达到所要求的价位时执行。

止损单

当市场价格达到预设价位时自动执行买或卖的定单。当市场价格达到定单价格，止损单将成为市价单，以市场当时最有利且可交易价位执行。请谨记，如果市场没有在定单的价格上交易，止损单可能由于市场跳空或滑点的影响而在定单设置的价格外执行。止损单将在定单价格被触发后，按照市场下一个最有利且可执行的价位成交。

止损入场单

在当前价位以上设置的买入定单，或在当前价位以下的卖出定单。该种定单可以在您认为市场将朝一个方向波动，而希望在某价位入场的情况下使用。

止损平仓单

这是一种设置以低于当前价格卖出（来平仓多头头寸），或以高于当前价格买入（来平仓空头头寸）的定单。止损定单是一种重要的风险管理工具。如果市场走势不利于您，则您可通过为持仓头寸设置止损定单限定您的可能损失。请记住，止损定单并不保证定单的执行价格 ,当止损单被触发，定单将按照市场最有利且可行的价位执行。

*如您需查看指定时间段的交易报告，请联系客户专员。

移动止损

移动止损定单可以自动调整触发交易执行的止损点价位，使之朝着有利的市场方向以一定幅度移动。在市场价格向有利方向移动的前提下，移动止损功能可以扩大盈利空间，并且一旦市场价格朝不利方向发展，交易可以自动完结。当市场价格朝有利方向发展（在价格上涨时持有多头头寸，或在价格下跌时持有空头头寸），止损点价位跟随着市场价格以一定幅度移动。如果市场价格朝不利的方向发展，则止损点价位保持不变，它和市场价格之间的差值也会缩小。如果市场价格继续朝不利的方向移动，并最终达到了设定的止损价位，那么交易将被执行。

以上所有类型的定单均为＂周末前有效定单＂（GTF），也就是周末收市前订单如果没有被执行就会被取消。定单在被执行或取消前都是有效的。如果您手动清算一笔头寸，必须取消所有与此头寸相关联的定单。须注意，条件定单有可能不能限制您的损失。

定单范围

黄金的止盈或止损单必须离当前市场价最少200点以外的距离。

白银的止盈或止损单必须离当前市场价最少20点以外的距离。

有效期

周末前有效定单（GTF），也就是周末收市前订单如果没有被执行就会被取消。

文传金银在北京时间周六收市后到周一开市前不执行交易。重新开市后，以开盘价而非我们交易台关闭时可能出现的价格予投资者进行交易。

盈亏计算

盈亏计算公式：( 卖出价 - 买入价 ) x 合约单位 - 利息 - 佣金

例子1：

李先生经网上平台买入2手伦敦金，即200盎司， 投入保证金2,000美元，买入价为1,300美元，并于即日以1,330美元卖出2手伦敦金平仓，他的盈利计算方式是：

投入保证金金额＝ 2,000美元

淨盈亏＝ (1,330 - 1,300) x 200 = 6,000美元

例子2：

陈小姐经网上平台下单买入1手伦敦金，即100盎司，投入保证金1,000美元，买入价为1,250美元，其后于第二天以1,220美元卖出1手伦敦金平仓，她的亏损计算方式是：

投入保证金金额＝ $1,000美元

持仓利息＝ 1,250 x 100 x 1.5% x (1/360) = 5.21美元

淨盈亏＝ (1,220 - 1,250) x 100 - 5.21（利息） = -3,005.21美元

注意事项

1.周末及假期休市前半小时会调整点差，以系统显示为准；行情波动大时，点差费用可能根据市况调整，报价以系统显示为准。

2.如周五及假期收市时，账户保证金不足100%，系统将会把所有仓位做锁仓（对冲）处理，以控制投资者的交易风险。

3.当账户的保证金比例不足20%时，系统会强制平掉所有的仓位。当价格波动大时，可能在低于20%的保证金比例时强制平仓。

4.周末或休市后所有未建仓的挂单及未执行的止损获利设置将全部取消。

5.交易时间将因应国际金融假期而作出相关调整，详情请留意本网站通告。
